OS 4 PILARES DA PROGRAMAÇÃO

Encapsulamento

Consiste em esconder os detalhes internos de um objeto e expor apenas o que é necessário para o uso externo.


Abstração

Tem como funcão simplificar o que é complexo em um sistema, destacando apenas o que for essencial


Herança

Concede uma nova classe que herda características de uma classe já existe, o que facilita o reuso de códigos.


Polimorfimo

Possibilita objetos de classes diferentes serem tratados como objetos de uma classe comum, normalmente usando métodos com o mesmo nome
